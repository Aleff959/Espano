# Espano

Fork do Espanso para navegadores — expansor de texto para Edge/Chrome sem servidor nem conta.

[![Version](https://img.shields.io/badge/versão-1.0.2-brightgreen)](releases)
[![License](https://img.shields.io/badge/licença-GPL3-red)](LICENSE)

</div>

---

Digite um trigger em qualquer campo de texto e ele é substituído automaticamente.

```
:email:  →  meu@email.com
:att:    →  Atenciosamente,
:obg:    →  Obrigado!
```

## Instalação

1. Baixa o `.crx` em [Releases](releases)
2. Abre `edge://extensions` → ativa **Modo de desenvolvedor**
3. Arrasta o `.crx` para a página

## Uso

Clica no ícone na barra do navegador → **＋ Nova** → define trigger e expansão → salva.

## Privacidade

Sem servidor. Dados salvos localmente via `chrome.storage.sync`.
